# 测试用例：发送时追加剪贴板内容

## 目标
验证启用“发送时追加剪贴板内容”后，发送提示词会在末尾拼接当前剪贴板文本；禁用时行为不变。

## 前置条件
- 启动应用并完成初始化。
- 已配置提示词目录且存在至少一条提示词。
- 系统剪贴板可用。

## 测试步骤与预期

### 场景 1：开关关闭（默认）
1. 在系统中复制一段文本（例如：`这是要生图的文段`）。
2. 打开启动器，选择任意提示词并发送（回车或点击）。
3. 预期：发送内容仅为提示词文本，不包含剪贴板内容。

### 场景 2：开关开启 + 剪贴板有内容
1. 打开设置，启用“发送时追加剪贴板内容”。
2. 在系统中复制一段文本（例如：`这是要生图的文段`）。
3. 发送任意提示词。
4. 预期：发送内容为“提示词文本 + 分隔符 + 剪贴板文本”。
   - 分隔符为：`\n\n---\n\n`。

### 场景 3：开关开启 + 剪贴板为空
1. 打开设置，启用“发送时追加剪贴板内容”。
2. 清空剪贴板（或复制空字符串）。
3. 发送任意提示词。
4. 预期：发送内容仅为提示词文本，不报错。

### 场景 4：鼠标与快捷键一致
1. 开关开启，剪贴板有内容。
2. 通过鼠标点击结果发送一次。
3. 通过快捷键发送一次。
4. 预期：两次发送内容一致，均追加剪贴板文本。
