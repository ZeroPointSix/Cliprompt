# TC-startup-hotkey

## 目标
验证开机自启动场景下：全局快捷键可用、窗口可见、搜索输入框可立即输入。

## 前置条件
- 已启用“开机自启动”。
- 热键为默认 `Alt+Space` 或已设置的新热键。
- 应用已退出，确保下次开机为冷启动。
- 如提示 `HotKey already registered`，请先在设置中更换为未被占用的热键。

## 用例 1：自启动后第一次唤起
1. 重启系统并登录。
2. 等待桌面加载完成后，按下全局快捷键。
3. 观察窗口是否可见并能输入搜索词。

预期结果：
- 窗口被正常唤起且不白屏。
- 搜索框自动聚焦，可直接输入并触发搜索结果。

## 用例 2：前端未就绪时的唤起排队
1. 重启系统并登录。
2. 桌面刚出现后立即按下全局快捷键（尽量早于应用完成初始化）。
3. 观察窗口是否在稍后自动显示并可搜索。

预期结果：
- 若前端尚未就绪，唤起会被延迟。
- 前端就绪后窗口自动显示，搜索框自动聚焦。

## 用例 3：热键切换回归
1. 打开设置，修改为新热键并保存。
2. 关闭窗口，使用新热键唤起。
3. 关闭窗口，使用旧热键验证不可用。

预期结果：
- 新热键可唤起窗口并聚焦输入框。
- 旧热键不会唤起窗口。
